# Telegram_gRaBber

Account-Level Risks
Telegram account ban — Telegram may flag the account for unusual activity (downloading hundreds/thousands of files rapidly). This is the biggest risk. If they detect bulk downloading, they may temporarily or permanently ban the account.
Rate limiting — Telegram will slow you down with FloodWaitError if you go too fast. The script has a 0.5 second delay to help with this, but for very large groups it can take hours.
Phone number restriction — If the account gets banned, the phone number may be restricted from creating new Telegram accounts for a period.
